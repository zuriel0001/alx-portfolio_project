Rock Paper Scissors

About the Game About Us

This is a classic game where players choose between rock, paper, or scissors, and the winner is determined based on the choices made and how the game rules apply to these choices.

About the Game

This is a detailed description of the game...

The home page has a description of the game and how it could be played. The game rules are also on the home page as well as the options to choose which opponent to play with. Either against the computer or against a friend.

When you make a choice, the computer does the same at the same time. Then the rules are applied, and the winner is determined. The scores are updated after each round, and the final winner is displayed in a popup at the end of the 5th round. You can always RESET and play again.

To play with a friend, you must share a unique game link with them, and both parties must open the link in their browsers. To get the link, click on COPY LINK; this command generates and copies the link at the same time.
